# Integrated Regression and Classification Modeling for Operational Insight and Process Optimization in Chicago from 2014 to 2024 
A collaborative effort between Mason Colt, Louis Finder and Liam O'Donnell, we looked to examine how different ML techniques could provide insight into permit approval in Chicago from 2014-2024. In this analysis we utilized multiple linear regression, logistic regression, ridge regression, natural cubic splines, k nearest neighbors (kNN) clustering, and naive Bayes clasification. 

To render the dashboard, open the file in RStudio and select knit -> knit to flexdashboard. Please ensure all libraries are installed, we recommend running each code chunk sequentially to diagnose what is not installed.
